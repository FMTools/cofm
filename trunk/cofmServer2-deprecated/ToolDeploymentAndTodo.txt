---------------------------------
Deploy the server via source code
---------------------------------
1. Create a database named "cofm" in MySQL 5.
2. Run "mvn-compile.bat".
3. Run "mvn-init-db.bat" (for the first time only).
4. Run "mvn-run-server.bat" to start the server.
